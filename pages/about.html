Football Viewer

Football Viewer adalah website yang menyediakan informasi seputar sepak bola internasional kamu mengakses web ini jika ingin mengetahui informasi jadwal pertandingan, hasil classement dan team yang bermain pada setiap league. Aplikasi ini juga berbasis PWA yang artinya kamu dapat mengaksesnya walaupun tidak ada sinyal iternet dengan syarat saat pertama kali mengakses website ini kamu harus tergubung terlebih dahulu k internet untuk meminta data kepada server dan di chace di perangkatmu. kamu juga dapat menambahkan website ini ke halaman beranda seperti aplikasi native lainnya.

Jika kamu tidak paham terhadap aplikasi berbasis PWA, kamu dapat mencarinya di internet pengertia tersebut. aplikasi ini juga menggunakan framework Materialize css.

Website berbasis PWA sangat diminati banyak orang pada saat ini dikarenakan dapat memberikan pengalaman yang mengesankan karena pengguna bisa menggunakan website seperti menggunakan aplikasi native yang ada pada smartphone. Sejak awal website ini bertujuan untuk memuaskan dan memenuhi keinginan para pemakai perangkat mobile sehingga dibuatlah website ini menggunakan PWA.
